# restaurant-page

This project was built as part of **The Odin Project** curriculum.

It is a simple restaurant website created using **JavaScript modules** and bundled with **Webpack**. The main goal of this project was to practice:

- Setting up a project with Webpack
- Organizing code into separate modules
- Dynamically generating DOM content with JavaScript
- Implementing tab-based navigation without reloading the page

The website includes three main sections:

- Home
- Menu
- Contact

Each section is generated entirely with JavaScript and rendered inside a single `#content` div. Navigation buttons dynamically switch between tabs by clearing and reloading the appropriate module content.
